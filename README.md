# Интеллектуальный анализ данных

Репозиторий содержит практические работы, выполненные в рамках курса **"Интеллектуальный анализ данных"** на 2-3 курсе обучения в НИУ ВШЭ. Курс разделён на две части: классические методы машинного обучения и основы глубинного обучения.

## Содержание

### Часть 1. Машинное обучение

| Ноутбук | Тема | Описание |
|---------|------|----------|
| **hw01_numpy.ipynb** | Основы NumPy | Работа с многомерными массивами, индексация, векторизованные операции, линейная алгебра |
| **hw02_pandas.ipynb** | Основы Pandas | Загрузка и обработка табличных данных, группировки, агрегации, работа с пропусками |
| **hw_03_EDA.ipynb** | Разведочный анализ данных (EDA) | Визуализация распределений, анализ корреляций, выявление выбросов, первичный анализ признаков |
| **hw_04_KNN_Linreg.ipynb** | KNN и линейная регрессия | Реализация и применение методов k-ближайших соседей и линейной регрессии, оценка качества |
| **hw_05_GradientDescend.ipynb** | Градиентный спуск | Реализация градиентного спуска, оптимизация функций, влияние learning rate на сходимость |
| **hw_06_TextsClassification.ipynb** | Классификация текстов | Векторизация текстов (TF-IDF), построение классификаторов для обработки естественного языка |
| **hw_07_RandomForest.ipynb** | Случайный лес | Построение композиций решающих деревьев, анализ важности признаков, подбор гиперпараметров |
| **hw_08_Boosting_Clustering.ipynb** | Бустинг и кластеризация | Градиентный бустинг (XGBoost/LightGBM) и алгоритмы кластеризации (K-means) |

### Часть 2. Глубинное обучение (Deep Learning)

| Ноутбук | Тема | Описание |
|---------|------|----------|
| **hw1_FullyConnectedNetwork.ipynb** | Полносвязные нейронные сети | Реализация многослойного перцептрона, обратное распространение ошибки, обучение на табличных данных |
| **hw2_Images.ipynb** | Компьютерное зрение | Свёрточные нейронные сети (CNN), классификация изображений, аугментация данных |
| **hw_03_Detection.ipynb** | Детекция объектов | Задачи обнаружения объектов на изображениях, архитектуры для детекции (YOLO) |
